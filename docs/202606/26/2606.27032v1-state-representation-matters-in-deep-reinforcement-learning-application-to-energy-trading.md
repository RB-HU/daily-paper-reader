## 摘要
能源交易决策不仅取决于当前市场价格，还取决于预期的未来市场状况和操作约束。这使得提供给强化学习智能体的状态表示成为一个重要的设计选择。我们在HydroDam（一个抽水蓄能套利环境）中使用固定的Double DQN智能体对此进行研究。环境、动作空间、奖励函数、网络和训练协议保持固定；仅改变市场特征。我们比较了绝对价格/日历特征、将当前价格与近期市场历史进行比较的相对特征、预测特征，以及这三种特征族的所有组合。策略使用2007-2011年比利时日前价格进行训练和选择，并在两个测试设置上进行评估：2012-2025年的后期相同市场测试集和39个其他ENTSO-E市场区域。仅使用绝对特征在测试集上达到28.8%，跨区域中位数为5.7%。仅相对和仅预测状态在跨区域中位数上也低于滚动价格评分启发式方法。组合特征族的效果更好：绝对+相对在测试集上达到49.9%，跨区域中位数为39.8%，而绝对+相对+预测达到55.6%和47.5%。这些结果表明，在储能交易强化学习中，状态表示不是次要的预处理选择，而是策略设计的核心部分：稳健的迁移需要结合价格尺度、近期相对价格背景和短期预测信息，而不是依赖任何单一特征族。

## Abstract
Energy trading decisions depend not only on current market prices, but also on expected future market conditions, and operational constraints. This makes the state representation given to a reinforcement learning agent an important design choice. We study this in HydroDam, a pumped-storage arbitrage environment, using a fixed Double DQN agent. The environment, action space, reward function, network, and training protocol are kept fixed; only the market features are changed. We compare absolute price/calendar features, relative features that compare current prices with recent market history, forecast features, and all combinations of these three feature families. Policies are trained and selected using 2007--2011 Belgian day-ahead prices and evaluated on two test settings: a later same-market test set from 2012--2025 and 39 other ENTSO-E market zones. Absolute features only reaches 28.8% on the test set and a median 5.7% across zones. Relative-only and forecast-only states also stay below a rolling price-score heuristic in the cross-zone median. Combining feature families is much stronger: absolute + relative reaches 49.9% on the test set and a 39.8% cross-zone median, while absolute + relative + forecast reaches 55.6% and 47.5%. These results suggest that state representation is not a minor preprocessing choice in storage-trading RL, but a central part of the policy design: robust transfer requires combining price scale, recent relative price context, and short-horizon forecast information, rather than relying on any single feature family.

## 2. 方法论

- **核心思想**：固定环境（HydroDam抽水蓄能套利环境）、动作空间（泵水/保持/放水）、奖励函数、DDQN网络架构，仅改变状态表示中的市场特征，进行受控消融研究，以孤立状态表示的影响。
- **HydroDam环境**：
  - 上水库容量100,000 m³，初始半满。
  - 最大泵水/放水速率18,000 m³/h，泵水效率0.8，发电效率0.9，水头差30m。
  - 每小时电价 `p_t` 既可为正也可为负，允许在负电价时泵水盈利。
  - 奖励函数为即时交易收益：
    \[
    r_t = \begin{cases}
    -p_t \Delta E_t / 0.8, & \text{泵水} \\
    0.9 p_t \Delta E_t, & \text{放水} \\
    0, & \text{保持}
    \end{cases}
    \]
    其中 \(\Delta E_t\) 为水体积变化对应的能量差。
  - 环境简化：无自然来水、无维护成本、无市场影响等。

- **状态表示设计**（所有状态均包含归一化水库容量）：
  - **绝对特征**：归一化价格（训练集99.9分位数）、对数价格、周末指示符、小时/星期正弦余弦编码。
  - **相对特征**：基于近期窗口（24h和168h）的价格z分数、滚动分位数价格评分（类似启发式基线）、相对日历特征（比较当前日历情境与近期同类情境的价格表现）、其他技术指标（价格区间位置、价格移动平均比、RSI、布林带位置）。
  - **预测特征**：由5个LightGBM模型集成生成的递归24小时价格差预报，包含未来价格z分数和预报不确定性。预报器本身使用了相对特征作为输入。

- **DDQN智能体**：
  - 网络结构：全连接，隐藏层大小1024-1024-512，LeakyReLU激活。
  - 训练参数：Huber损失，Adam（学习率10⁻⁴），折扣因子0.99，经验回放，目标网络每2500步更新，批量大小2048，ε-greedy探索（ε从1.0开始，每episode乘以0.95直到0.05）。
  - 所有策略均使用相同超参数和网络。

## 3. 实验设计

- **数据集**：
  - 训练/验证：比利时Belpex日前电价，2007–2009用于训练（26,304小时），2010–2011用于验证（17,520小时）。
  - 测试集1（同市场晚时段）：比利时2012–2025日前电价（122,736小时），包含负电价和能源危机时段。

- **Benchmark**：**滚动价格评分启发式** — 使用过去34小时价格计算第16和第68百分位数，构建评分：
  \[
  s_t = \frac{p_t - q_{16,t}}{q_{68,t} - q_{16,t}}
  \]
  当 \(s_t \le 0\) 泵水，\(s_t \ge 1\) 放水，否则保持。参数只在训练集上选定，全程固定。
- **对比方法**：
  - 7种DDQN状态表示消融：absolute、relative、forecast、absolute+relative、absolute+forecast、relative+forecast、absolute+relative+forecast。
  - 每种表示5个随机种子，基于验证集最优收益率（占各数据集理论最优利润百分比）选择检查点。
- **评价指标**：**策略所获奖励占该数据集理论最优利润的百分比**（最优利润通过已知未来价格的动态规划求得）。

## 6. 主要结论与发现

- **单特征族表现差**：absolute在验证集达78.9%，但在同市场测试集骤降至28.8%，跨区域中位数仅5.7%；relative和forecast在跨区域中位数也都低于启发式基线（29.2%）。
- **组合特征大幅提升**：
  - absolute+relative：测试集49.9%，跨区域中位数39.8%。
  - absolute+forecast：测试集49.5%，跨区域中位数40.0%。
  - absolute+relative+forecast：测试集55.6%，跨区域中位数47.5%，并且在平均39个区中有38.2个区胜过启发式。
- **相对特征与预测特征的互补性**：预测块本身已包含部分相对信息（预报器训练用相对特征），但直接给RL智能体提供相对特征仍能进一步改善效果；预测不能完全替代直接相对上下文。
- **状态表示是策略设计的核心**：单靠强算法（DDQN）不能自动弥补状态信息的缺失，合适的特征组合才能让RL策略在变化的市场中保持竞争力。

## 7. 优点

- **受控消融设计**：将环境、算法、训练流程固定，仅变化状态表示，直击核心问题，归因清晰。
- **多维度泛化测试**：不仅在同市场晚时段测试，还在39个欧洲市场区域进行零样本评估，全面考察迁移鲁棒性。
- **与强启发式基线比较**：滚动价格评分启发式简单且与任务目标高度适配，提供了有意义的性能下界。
- **特征工程的可解释性**：分解为三大特征族，阐明各自的优势和局限，为实际系统设计提供指导。
- **代码开源**，有利于复现和后续研究。

## 8. 不足与局限

- **单一环境与算法**：仅在HydroDam这一特定储能环境和Double DQN一种架构上验证，结论能否推广到其他存储设施（如电池、需求响应）或其他RL算法（如策略梯度方法）尚待检验。
- **种子数有限**：每种特征组合仅用5个种子，可能不足以捕获训练随机性带来的全部方差，尤其在验证集选优时存在一定选择偏差风险。
- **跨市场评估的时间对齐问题**：各区使用最长连续可用数据，年代分布不完全一致，可能使某些区域因包含特定事件（如能源危机）而与其他区域不可直接比较。
- **预报器与相对特征的信息重叠**：预测模型本身训练用了相对特征，使得“仅预测”的策略并非完全独立于相对信息，解释上需谨慎。
- **未涉及在线学习或微调**：所有策略均离线训练和固定，未评估策略在目标市场进一步微调后的性能提升。
- **现实建模简化**：未考虑水头变化、水流限制、效率曲线、维护成本、电力市场规则等更现实的物理和经济学因素，结论在更精细模型中的适用性有待验证。
