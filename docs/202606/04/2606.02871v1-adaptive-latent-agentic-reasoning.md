## 摘要
大型推理模型通过生成扩展的思维链（CoT）推理来提升性能，但这种行为在应用于LLM智能体时效率低下。当前的LLM智能体通常在每一步决策时生成冗长的文本推理，且几乎均匀地分配推理努力在各轮次，导致多轮智能体轨迹中的显著低效。我们提出自适应潜空间智能体推理（ALAR），一种双模式框架，在常规轮次中使用紧凑的潜空间推理，并在需要更深思熟虑时有选择地升级到显式思维链。ALAR通过将智能体的动作作为监督锚点来学习潜空间推理，并进一步优化以在潜空间推理足以完成任务时使用它，同时为更困难的决策保留显式CoT。在智能体搜索和工具使用基准上的实验显示，ALAR保持了可比或更好的任务准确率，同时显著减少了生成令牌，搜索中最多减少43.6%，工具使用中最多减少84.6%。这些结果表明，ALAR通过减少不必要的文本推理，同时为更困难的决策步骤保留显式深思熟虑，改善了LLM智能体的准确率-效率权衡。

## Abstract
Large reasoning models improve performance by generating extended chain-of-thought (CoT) reasoning, but this behavior becomes inefficient when applied to LLM agents. Current LLM agents often generate verbose textual reasoning at every decision step and allocate reasoning effort nearly uniformly across turns, leading to substantial inefficiency in multi-turn agentic trajectories. We propose Adaptive Latent Agentic Reasoning (ALAR), a dual-mode framework that uses compact latent reasoning for routine turns and selectively escalates to explicit chain-of-thought when deeper deliberation is needed. ALAR learns latent reasoning by using the agent's actions as supervision anchors and is further optimized to use latent reasoning when it is sufficient for task success and reserve explicit CoT for harder decisions. Experiments on agentic search and tool-use benchmarks show that ALAR maintains comparable or better task accuracy while substantially reducing generated tokens by up to 43.6% in search and 84.6% in tool use. These results demonstrate that ALAR improves the accuracy-efficiency trade-off of LLM agents by reducing unnecessary textual reasoning while preserving explicit deliberation for harder decision steps.

### 2. 方法论

论文提出了**自适应潜空间智能体推理（ALAR）**框架，其核心思想是为智能体提供两种推理模式，并通过两阶段训练学习何时使用哪种模式。

- **双模式推理架构**：
    - **潜空间推理模式**：作为默认模式，用于处理常规轮次。它不生成文本序列，而是在模型的隐藏状态空间中，通过一个投影层 `fϕ` 将一个固定长度的连续嵌入序列 `zt`（潜空间块）作为内部计算，然后基于此生成行动 `at`。
    - **显式推理模式**：保留传统的 CoT 能力。仅在需要深度规划、信息融合或反思的困难轮次中，模型自主选择生成文本思维链 `ct`，然后基于此生成行动 `at`。
    - **模式选择**：智能体在每轮开始时，直接从策略 `πθ` 中采样一个模式标记 `mt` （ `<LAT>` 或 `<THINK>` ），使模式选择成为其决策过程的一部分，而不是外部路由。

- **两阶段训练流程**：
    1.  **第一阶段：动作锚定自蒸馏（AASD）**，目标是在不失性能的前提下，教会模型使用潜空间推理模式。
        - **核心创新**：规避了直接监督潜空间状态的难题。它以同一基座模型作为教师（显式模式）和学生（潜藏模式），用教师模型在成功轨迹中产生的**动作为锚点**进行监督。
        - **训练过程**：首先用教师模型（显式模式）在任务中采样，保留成功的行动序列 `τa`。然后将学生模型中的每个教师 CoT 块替换为潜空间块 `<LAT>` + `zt` 。
        - **损失函数**：最大化在给定状态 `st` 和潜空间块 `zt` 的条件下，复现教师锚点动作 `at` 的对数似然，公式如下：
          `LAASD = -Eτa [ Σt ( log πθ(mt|st) + log πθ(at|st, mt, zt) ) ]`
          该损失反向传播通过 Transformer 和投影链，实现了对潜空间块的端到端学习。
    2.  **第二阶段：自适应推理 GRPO (AR-GRPO)**，目标是在 AASD 的基础上，学习如何自适应地选择推理模式。
        - **核心思想**：通过强化学习优化完整轨迹的模式选择策略，鼓励在保持任务成功的前提下使用潜藏模式，并在困难轮次保留显式推理能力。
        - **轨迹奖励设计**：
            - **非对称格式奖励 `rfmt`**：对于成功的轨迹，其潜藏模式使用比例 `f(τ)` 越高，奖励越大；对于失败的轨迹，其潜藏模式使用比例越高，惩罚越重。
              `rfmt(τ) = 1 + αf(τ)` （若成功）； `rfmt(τ) = -αf(τ)` （若失败）。
            - **衰减多样性奖励 `rdiv`**：在早期鼓励探索不同模式配比，后期逐渐衰减，让格式奖励主导。
            - **长度缩放因子 `sL`**：对超出容忍长度的显式 CoT 进行惩罚。
        - **优化**：使用标准的 GRPO 裁剪目标函数和 KL 散度惩罚，对完整轨迹 `τ` 进行优化，更新策略 `πθ`。
