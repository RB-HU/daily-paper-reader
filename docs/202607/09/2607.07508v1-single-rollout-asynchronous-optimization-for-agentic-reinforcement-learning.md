## 摘要
强化学习（RL）在大语言模型（LLM）的后训练中变得越来越重要。以往的LLM强化学习流程大多是同步且批次交错的，这对于长周期智能体任务效率低下。最近，异步强化学习作为一种更高效的替代方案出现，能够在推演到达时立即更新模型。然而，现有的异步强化学习系统往往强调吞吐量，而训练稳定性和任务有效性在很大程度上未被充分探索。例如，一个关键挑战是广泛使用的GRPO框架中的分组采样并不能自然地适应异步智能体训练。在本文中，我们提出了单次推演异步优化（SAO）来解决异步强化学习中的稳定性和离策略挑战。为了减少离策略影响并提高泛化能力，我们用单次推演采样取代分组采样，即每个提示仅使用一次推演。我们通过实用的价值模型训练设计进一步改进了这种单次推演策略。为了提高优化稳定性，我们引入了一种严格的双向令牌级裁剪策略。SAO能够稳定训练上千步，并在智能体编码和推理基准测试（如SWE-Bench Verified、BeyondAIME和IMOAnswerBench）上持续优于GRPO及其变体。我们还证明，单次推演强化学习在模拟在线学习环境中特别有效，因为模型必须适应不断变化的环境。为此，SAO已成功部署到用于训练开源GLM-5.2模型（750B-A40B）的智能体强化学习流程中。

## Abstract
Reinforcement learning (RL) is becoming increasingly important for post-training large language models (LLMs). Previous RL pipelines for LLMs were mostly synchronous and batch-interleaved, which is inefficient for long-horizon agentic tasks. Recently, asynchronous RL has emerged as a more efficient alternative by updating the model as rollouts arrive. However, existing asynchronous RL systems often emphasize throughput, while leaving training stability and task effectiveness largely underexplored. For example, a key challenge is that group-wise sampling in the widely-used GRPO framework does not naturally fit asynchronous agentic training. In this paper, we present Single-rollout Asynchronous Optimization (SAO) to address the stability and off-policy challenges in asynchronous RL. To reduce off-policy effects and improve generalization, we replace group-wise sampling with single-rollout sampling, that is, using one rollout per prompt. We further improve this single-rollout strategy with practical value-model training designs. To improve optimization stability, we introduce a strict double-side token-level clipping strategy. SAO is able to train stably for one thousand steps and consistently outperform GRPO and its variants on agentic coding and reasoning benchmarks, such as SWE-Bench Verified, BeyondAIME, and IMOAnswerBench. We also demonstrate that single-rollout RL is particularly effective in a simulated online learning setting, where the model must adapt to changing evolving environments. To this end, SAO is successfully deployed in the agentic RL pipeline for training the open GLM-5.2 model (750B-A40B).

*   **研究动机**：大语言模型（LLM）后训练正转向强化学习（RL），但现有RL流程大多为同步、批次交错模式。对于长时间运行的智能体（Agentic）任务（如代码修复、数学推理），推演长度差异巨大，同步模式会导致大量GPU算力因等待最长推演而闲置，效率低下。
*   **核心问题**：异步RL通过推演完成即更新的方式提升了效率，却引入了两大挑战：
    *   **训练不稳定**：推演由多个版本的旧模型产生，导致严重的、不可预测的离策略（off-policy）问题。
    *   **算法不匹配**：当前主流的GRPO等算法依赖分组采样（同一提示生成一组响应，组内比较），这与异步训练天然矛盾（需等待组内所有推演完成），且不适合多数仅提供单条轨迹的在线环境。
*   **整体含义**：本文旨在解决异步RL训练中的稳定性与离政策漂移问题，使其在保持高效的同时，能够有效提升模型在智能体任务上的表现。

### 2. 论文提出的方法论

SAO通过三个层面的设计，构建了一个稳定的单次推演异步RL框架：

*   **核心技术一：直接双向重要性采样（DIS, Direct Double-sided Importance Sampling）**
    *   **核心思想**：为应对异步训练中无法追踪准确历史策略的问题，直接使用推演引擎记录的log-probabilities作为行为策略`π_rollout`，与当前策略`π_θ`计算重要性比率，彻底抛弃了传统的旧策略模型`π_θold`。
    *   **严格双端裁剪与掩码**：为进一步稳定训练，引入严格的双向token级裁剪策略。将重要性比率`rt(θ) = exp(log π_θ - log π_rollout)`限制在`[1-ϵ_l, 1+ϵ_h]`区间内，任何超出此区间的token将被完全掩码，不参与梯度计算。这比标准PPO/GRPO的裁剪更为激进，有效防止了极端策略发散。

*   **核心技术二：单次推演采样与价值模型增强**
    *   **核心思想**：用“每提示单次推演”替代GRPO的“分组采样”，推演一旦完成立即送入训练，从根本上解决了组内等待问题。为解决单次推演带来的高方差问题，SAO设计了一系列工程化的价值模型（Critic）训练策略：
    *   **价值网络快速更新**：每更新一次策略（Actor），价值网络更新K次（论文中K=2），确保价值估计能更快跟上策略变化，提供更准确的优势估计。
    *   **冻结注意力参数**：仅训练价值模型的MoE（混合专家）层，冻结自注意力层参数。实验发现这能显著降低价值模型的梯度范数，提升训练稳定性。
    *   **跳过观察的Token级GAE**：针对包含环境反馈的多轮智能体轨迹（Action→Observation→Action），专门设计了优势估计器。它在计算时序差分（TD）误差和广义优势估计（GAE）时，会“跳过”环境反馈的观察tokens，直接链接当前动作的末尾token与下一个动作的起始token。公式为：`δ = r_t + γV(a_{i+1,0}) - V(a_{i,N})`，`Â(a_{i,N}) = δ + γλÂ(a_{i+1,0})`。这避免了将噪声引入基于模型输出的优势估计中。
