## 摘要
智能体AI系统通过部署能够在网络边缘进行协作学习同时保持数据隐私的自主智能体，正在重塑通信和网络。在分布式网络环境中，多模态大语言模型（MLLMs）作为边缘设备的认知引擎，然而在异构网络条件下，联邦微调在平衡全局知识聚合与局部自适应方面面临巨大挑战。传统的联邦协议通常依赖于统一的参数聚合，将域不变特征与客户端特定细微差别混为一谈，从而导致次优的个性化以及过高的通信开销。为解决这些挑战，我们提出了PFAdapter，一个通信高效的框架，引入层次化LoRA分解，将适配器参数显式地分离为全局共享和局部私有两部分。查询和键投影被分配给全局同步，以捕获跨网络的通用多模态语义，而值和输出投影则保留在本地用于边缘特定适应。此外，基于Frobenius范数的正交正则化强制这些组件之间的严格分离，防止冗余特征学习。选择性聚合协议仅同步整个联邦网络中的全局共享组件，保留本地专长，并将通信成本降低近50%。在VQA-RAD、SLAKE、Hateful Memes和CrisisMMD数据集上的大量实验表明，PFAdapter在不同边缘智能任务中始终优于最先进的基线，准确率提升幅度从2.4%到4.8%不等。因此，我们的框架为资源受限通信网络中的智能体AI部署建立了一种高效的解决方案。

## Abstract
Agentic AI systems are reshaping communications and networking by deploying autonomous intelligent agents capable of collaborative learning while maintaining data privacy at network edges. Within distributed network environments, Multimodal Large Language Models (MLLMs) serve as cognitive engines for edge devices, yet federated fine-tuning faces substantial challenges in balancing global knowledge aggregation with local adaptation under heterogeneous network conditions. Conventional federated protocols typically rely on uniform parameter aggregation, which conflates domain-invariant features with client-specific nuances, thereby resulting in suboptimal personalization and excessive communication overhead. To address these challenges, we propose PFAdapter, a communication-efficient framework introducing hierarchical LoRA decomposition to explicitly separate adapter parameters into global-shared and local-private components. Query and key projections are assigned to global synchronization for capturing universal multimodal semantics across the network, while value and output projections remain localized for edge-specific adaptation. Additionally, orthogonality regularization based on the Frobenius norm enforces strict separation between these components, preventing redundant feature learning. Selective aggregation protocols synchronize only global-shared components across the federated network, preserving local expertise and reducing communication costs by nearly 50%. Extensive experiments on VQA-RAD, SLAKE, Hateful Memes, and CrisisMMD datasets demonstrate that PFAdapter consistently outperforms state-of-the-art baselines, achieving accuracy improvements ranging from 2.4% to 4.8% across diverse edge intelligence tasks. Consequently, our framework establishes an efficient solution for agentic AI deployment in resource-constrained communication networks.

    - **通信开销高昂**：即使采用参数高效微调(如LoRA)，每个通信轮次仍须传输全部适配器参数(数百MB)，对带宽受限的工业边缘网络难以承受。
- **本文目标**：设计一种通信高效的个性化联邦学习框架，通过细粒度的适配器分解，使全局共享的知识与本地私有的专长相互分离，在保证模型性能的同时将通信量降低近一半。

## 2. 方法论：分层LoRA分解与选择性聚合
- **核心思想**：基于自注意力模块中不同投影层的功能差异，将LoRA适配器参数划分为 **全局共享** 与 **本地私有** 两个集合。
- **参数分解策略**：
    - **全局同步 (ΘG)**：所有Transformer层的查询(Query, Wq)和键(Key, Wk)投影的LoRA增量(ΔWq, ΔWk)。
        - 依据：查询/键投影决定“哪些词元互相关注”，捕获跨域通用的结构对齐关系，适合在客户端间共享。
    - **本地保留 (ΘL)**：值(Value, Wv)和输出(Output, Wo)投影的LoRA增量(ΔWv, ΔWo)。
        - 依据：值/输出投影负责“注入具体的语义内容”，高度依赖客户端数据分布，必须保留在本地。
- **正交正则化 (Orthogonality Regularization)**：
    - 通过 **Frobenius范数** 构建正则项，强制全局与本地参数矩阵的列空间近似正交，防止本地组件冗余学习已被全局组件捕获的知识。
    - 损失函数：\( \mathcal{L}_o = \sum_{l} \sum_{m_g \in \{q,k\}} \sum_{m_l \in \{v,o\}} \| W_{l,m_g}^\top W_{l,m_l} \|_F^2 \)。
    - 总本地目标：\( \mathcal{L}_{\text{total}} = \mathcal{L}_{\text{task}} + \lambda_o \mathcal{L}_o \)。
- **选择性聚合与通信方案**：
    - 每个通信轮次，**仅上传并聚合**属于 ΘG 的查询/键适配器参数。
    - 值/输出适配器 ΘL 从不上传，仅在本地数据上更新并驻留于客户端。
    - 通信量减少：在rank相同、隐藏维度一致的条件下，传输参数精确减半（4个投影中只传2个），实际测量从617 MB/轮降至315 MB/轮。
- **算法流程 (Alg.1)**：
    1. 服务器初始化并分发全局的Q/K适配器，客户端初始化私有的V/O适配器。
    2. 每轮选中客户端在本地数据上进行多轮SGD更新，同时优化任务损失与正交损失。
    3. 每轮结束后，客户端仅将更新后的Q/K适配器发回服务器。
    4. 服务器按数据量加权平均Q/K适配器，V/O适配器继续保持客户端私有。
- **理论收敛保证**：在非凸平滑假设、有界异质性及有界聚合偏差下，框架收敛速率可达 \( \mathcal{O}(1/\sqrt{T}) \)。

## 7. 方法优点
- **细粒度知识分离**：首次在适配器内部（投影层级别）区分全局共享与本地私有参数，而非粗粒度的整层分割，更契合attention的运作机理。
- **通信量原则性减半**：仅需同步Q/K两组低秩矩阵，实现约50%的通信压缩，原理简洁且易于部署。
- **正交正则化**：利用Frobenius范数为全局‑本地组件提供可导的分离压力，避免了客户端只学冗余特征，并缓解了聚合时的“权重冲洗”。
- **即插即用**：与标准LoRA保持相同的内存和计算开销（仅增加极少量正交损失计算），不改变模型结构，便于在已有的MLLM和联邦生态中推广。
- **冷启动友好**：新加入客户端可直接利用已聚合的全局Q/K适配器，在少量本地步内快速适配，无需从头参与全联邦训练。

## 8. 不足与局限
- **固定的分解比率**：当前设置强制50:50分割，虽然对中等异构场景效果良好，但在更加极端或特殊的异构网络环境下，可能需要动态或可学习的分解比率。
- **均匀分层分配**：所有Transformer层使用相同的投影分配策略，未考虑浅层和深层对共享/私有知识的不同需求，层间差异化的分割可能进一步提升性能。
- **正交约束的复杂性**：基于Frobenius范数的线性正交在捕捉复杂的非线性依赖关系时可能能力有限；更精细的信息论或结构化解耦方法值得探索。
- **隐私讨论有限**：虽然论文明确声明其隐私保护仅限于“训练数据不集中”，并未防御梯度泄露、成员推断攻击等更强的威胁模型，在严格隐私合规场景下仍需集成差分隐私等机制。
- **场景覆盖**：目前验证集中在两个医学、两个社交数据集，对于其他高频边缘智能任务（如工业视频理解、自动驾驶）的泛化性仍需补充实验。
