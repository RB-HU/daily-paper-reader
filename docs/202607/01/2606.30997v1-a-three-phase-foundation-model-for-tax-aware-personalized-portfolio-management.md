## 摘要
我们提出了一种用于个性化投资组合管理的三相深度强化学习系统，解决了所有先前金融强化学习工作共有的三个局限：1）股票代码锁定，2）单一目标，以及3）静态用户模型。阶段一通过在多资产语料上的自监督学习预训练一个不依赖股票代码的跨资产编码器，并通过一个冻结的并行分支（使用基于T5的时间序列基础模型Chronos）进行增强，再通过一个可学习的门控机制进行融合。据我们所知，这是时间序列基础模型在投资组合管理强化学习中的首次应用。该编码器通过一个50维的可观测元数据向量泛化到任何公开交易的资产，对新代码无需重新训练。阶段二使用PPO对MoE（专家混合）投资组合的演员-评论家进行微调，并采用目标条件奖励，同时服务每回合采样的六个不同的投资目标：短期阿尔法、短期收益、长期收益、资本保值、税损收割和仅长期收益。MoE架构将每个目标分配给专门的专家头（动量、成长、防御、税务感知），并通过一个可学习的意图路由器，根据当前活跃目标和市场状态混合专家，从而消除跨目标的梯度冲突。阶段三增加了一个轻量级的个性化层，在推理时通过一个76参数的LoRA模块进一步适应每个个体，该模块在真实经纪交易历史上微调，从揭示的交易行为而非问卷中推断投资目标。一个自然语言意图解析器将自由形式的目标直接转换为结构化的投资目标参数。

## Abstract
We present a three-phase deep reinforcement learning system for personalized portfolio management that addresses three limitations shared by all prior financial RL work: 1) ticker lock-in, 2) monolithic objectives , and 3) static user models. Phase 1 pretrains a ticker-identity-free cross asset encoder via self-supervised learning on a multi-asset corpus, augmented by a frozen parallel branch using Chronos, a T5-based time series foundation model, fused via a learned gating mechanism. To our knowledge, this is the first application of a time series foundation model to portfolio management RL. The encoder generalizes to any publicly traded asset via a 50-dimensional observable metadata vector that requires no retraining for new tickers. Phase 2 fine-tunes a MoE (Mixture of Experts) portfolio actor critic with PPO under an objective-conditioned reward that simultaneously serves six distinct investment goals sampled per episode: short-term alpha, short-term gain, long-term gain, capital preservation, tax-loss harvesting, and long-term-gains-only. A MoE architecture assigns each objective to a specialized expert head (momentum, growth, defensive, tax-aware), and a learned intent router blends experts based on the active objective and current market regime, which eliminates cross-objective gradient conflict. Phase 3 adds a lightweight personalization layer further adapted at inference time to each individual via a 76-parameter LoRA module fine-tuned on real brokerage transaction history, inferring investment objectives from revealed trading behavior rather than questionnaires. A natural language intent parser converts free-form goals directly into structured investment objective parameters.

### 一、研究背景与核心问题
*   **核心问题**：现有金融强化学习系统存在三大根本性局限：
    1.  **股票代码锁定 (Ticker Lock-in)**：模型训练在固定的资产池上，无法泛化到新股票，必须重新训练。
    2.  **单一目标 (Monolithic Objectives)**：几乎所有策略都优化单一的夏普比率，无法同时满足用户多样化的投资目标（如税损收割、资本保值、短期阿尔法等）。
    3.  **静态用户模型 (Static User Models)**：偏好仅通过一次性问卷获取，从不动态更新，无法反映用户真实交易行为中隐含的变化。
*   **整体含义**：旨在构建一个突破上述限制的个性化投资组合管理系统，能够零样本泛化到任意公开资产，同时服务六种以上的投资意图，并从用户真实的交易历史中持续自适应其偏好。

### 二、方法论
论文提出“**三相基础模型**”，各阶段核心思想与技术细节如下：

*   **阶段一：自监督跨资产编码器预训练**
    *   **消除股票代码依赖**：完全摒弃固定的股票嵌入表，改用 **50维可观测元数据向量**（涵盖行业、基本面、分析师共识、期权信号、财报日程、内部人情绪、机构持仓等）。该向量可对任何公开资产计算，无需重训。
    *   **多模态时序编码**：
        *   主体为**市场编码器**（Transformer），基于价格特征窗口做自监督学习（下一个bar收益率预测、掩码特征恢复、市场状态分类）。
        *   并行引入冻结的 **Chronos-T5-Small 时间序列基础模型**（在1000亿以上数据点预训练），仅训练其投影头与一个可学习的门控机制，将通用时序表征与领域专用SSL表征融合。
    *   **关键创新——跨资产对比损失**：发现标准SSL会导致所有股票的表征坍缩为均值（余弦相似度 0.96）。引入跨资产对比损失最小化不同股票表征间的余弦相似度，降至 0.24，使得分配头能输出差异化的权重。

*   **阶段二：目标条件化的混合专家（MoE）投资组合策略**
    *   **混合专家架构**：定义四个专家分配头——**动量、成长、防御、税务感知**。每个专家专注一类目标（如动量专家服务短期阿尔法和30日收益，税务专家服务税损收割和仅长期收益）。
    *   **意图路由器**：接收市场状态和one-hot目标向量，输出专家混合权重。路由器可基于市场状态动态调整权重（如高波动时即使阿尔法目标也部分路由到防御专家），且只更新路由器，专家头可冻结，消除跨目标梯度冲突。
    *   **奖励塑形**：目标条件奖励由 **目标基座（如阿尔法、卡玛比率、含税收益等）** 加上惩罚项（集中度、净周转、现金拖累、停滞惩罚、重新部署奖励）构成。其中重新部署奖励鼓励 “卖-立刻买” 的闭环，并定义去除了往返交易的净周转。
    *   **其他关键设计**：**可学习的现金令牌**强制策略主动决定现金比例，避免“全持仓”陷阱；**分配驱动执行**：当目标权重偏离超过阈值则强制再平衡，防止动作头输出全“持有”阻塞交易。

*   **阶段三：基于LoRA的税务感知个性化**
    *   在真实经纪商交易历史上分析用户的显性行为（持有期、收割行为、处置效应等），构建16维用户画像。
    *   通过**仅76参数的低秩适配器（LoRA）** 微调意图路由器，实现个性化权重偏移，而无需触碰编码器或专家头。实现从“说出的偏好”到“揭示的偏好”的过渡。
    *   包含自然语言意图解析器，将自由文本目标映射为结构化参数。
